## Overview

A curated collection of **competitive programming solutions** covering problems from various online judges and coding competitions. This repository documents algorithmic problem-solving approaches, data structure implementations, and optimization techniques developed through practice and competition participation.

---

## Key Features

- **Diverse problem set** — Solutions spanning multiple difficulty levels and categories
- **Multiple platforms** — Problems from LeetCode, Codeforces, HackerRank, and more
- **Well-documented** — Clean code with approach explanations
- **Algorithm coverage** — Sorting, searching, graph theory, dynamic programming, greedy, and more

---

## Topics Covered

| Category | Concepts |
|---|---|
| Arrays & Strings | Two pointers, sliding window, prefix sums |
| Trees & Graphs | BFS, DFS, shortest path, MST |
| Dynamic Programming | Memoization, tabulation, state optimization |
| Greedy Algorithms | Activity selection, interval scheduling |
| Sorting & Searching | Binary search, merge sort, quicksort |
| Data Structures | Stacks, queues, heaps, hash maps, tries |
| Math & Number Theory | GCD, modular arithmetic, prime sieve |
